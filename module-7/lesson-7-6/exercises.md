# Exercises: Snapshot & Visual Testing

1. **Create Snapshot**: Write snapshot test for a component
2. **Multiple Variants**: Test different component variants
3. **Update Snapshots**: Intentionally change component and update snapshot
4. **Review Changes**: Review and approve snapshot updates
5. **Regression Detection**: Verify snapshot catches unintended changes

**Grading**: All snapshots created, updates managed properly, regressions detected
