# Sample Project: E2E Tests for Todo App

Create comprehensive E2E tests for the Todo application testing complete user workflows.

## Test Scenarios

1. **Add Todo**: Add todo item and verify display
2. **Complete Todo**: Mark todo as done and verify styling
3. **Delete Todo**: Delete todo and verify removal
4. **Complex Workflow**: Add multiple, complete some, delete some
5. **Filtering**: Filter todos by status

## Grading Criteria

✅ All tests pass  
✅ Each test is independent  
✅ Uses data test IDs  
✅ Tests complete workflows  
✅ Proper waiting/timing
