# Exercises: E2E Testing with Cypress

1. **Setup Cypress**: Install and configure Cypress for your Vue app
2. **Basic Test**: Write test to add and display todo item
3. **Form Validation**: Test form validation and error display
4. **Complete Workflow**: Test add → complete → delete sequence
5. **Navigation**: Test navigation across multiple pages
6. **API Integration**: Test workflows with real/mocked API calls

**Grading**: All tests pass, workflows covered
