# Lesson 1.6: Events & Emits

## Overview
Learn how child components communicate back to their parents using custom events. Master event emission, event payloads, v-on modifiers, and two-way data binding patterns.

**Duration**: 60-90 minutes  
**Prerequisites**: Lessons 1.1-1.5

---

## Learning Objectives

By the end of this lesson, you'll be able to:

✅ Emit custom events from child components  
✅ Pass data through event payloads  
✅ Listen to events with `@` directive  
✅ Use v-on modifiers effectively  
✅ Implement v-model for two-way binding  
✅ Understand event flow and bubbling  
✅ Build bidirectional component communication
