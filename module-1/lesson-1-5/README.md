# Lesson 1.5: Component Basics & Props

## Overview
Learn how to build reusable components and pass data between parent and child components using props. Master prop declaration, validation, type safety, and composition patterns.

**Duration**: 60-90 minutes  
**Prerequisites**: Lessons 1.1-1.4

---

- **[Quiz](./quiz.md)** - 30 questions to test your knowledge
- **[Quiz Answers](./quiz-answers.md)** - Detailed answer explanations
- **[Exercises](./exercises.md)** - 5 practical exercises

---

## Learning Objectives

By the end of this lesson, you'll be able to:

✅ Create reusable Vue components  
✅ Declare and validate props with TypeScript  
✅ Set default values and required props  
✅ Understand one-way data flow  
✅ Compose components effectively  
✅ Handle prop types correctly  
✅ Follow component communication best practices
