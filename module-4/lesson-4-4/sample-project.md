# Sample Project: Advanced App with Lazy Loading

**Module 4, Lesson 4.4**

---

Build an app demonstrating:

1. **Lazy Loading**: Admin routes lazy loaded
2. **Transitions**: Fade between pages
3. **Named Views**: Header + sidebar + main
4. **Dynamic Routes**: Add routes at runtime
5. **Error Handling**: Handle chunk load failures

**Time**: 25-30 minutes

## Key Features:
- Admin section in separate chunk
- Smooth page transitions
- Multi-panel dashboard layout
- Loading states for lazy routes
- Fallback for load errors
