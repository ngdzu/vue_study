# Sample Project: Product Catalog with Filters

**Module 4, Lesson 4.3**

---

## Build a product catalog with:
- Dynamic product detail pages
- Search functionality
- Category filters
- Pagination
- URL-based state

## Key Routes

```ts
'/products' - List with filters (query params)
'/products/:id' - Detail page (param)
```

## Implement:
1. Product list with search query param
2. Category filter in query params
3. Pagination with page query param
4. Product detail with ID param
5. Preserve filters when navigating back

**Time**: 25-35 minutes
