# Quiz Answers: Route Parameters & Query Strings

See quiz.md for questions and answers.

**Key Concepts:**

1. **Dynamic Params**: Use `:param` syntax
2. **Query Params**: Access via `route.query`
3. **Type Conversion**: Always validate and convert
4. **Props**: Use `props: true` for params
5. **Preservation**: Spread existing query params
6. **Validation**: Regex patterns for params
7. **Pagination**: Page number in query
8. **Search**: Query param with debounce
9. **Hash**: Scroll to sections
10. **Best Practice**: Params for required, query for optional
