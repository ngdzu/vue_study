# Quiz: HTML Forms Fundamentals

Select the best answer for each question.

1) What is the purpose of the `<label>` element?
A. Display text only
B. Associate text with form field; improves accessibility
C. Style form fields
D. Submit form data

2) Which input type validates email format automatically?
A. `type="text" pattern="[^@]+@[^@]+"`
B. `type="email"`
C. `type="mail"`
D. Any type with `required` attribute

3) What does the `required` attribute do?
A. Makes field bold
B. Browser prevents form submission if empty
C. Disables the field
D. Hides the field

4) Which attribute specifies a pattern validation rule?
A. `validate="regex"`
B. `regex="[A-Z]+"`
C. `pattern="[A-Z]+"`
D. `validation="pattern"`

5) What is the difference between GET and POST?
A. GET submits in URL (visible); POST in body (hidden)
B. POST submits in URL; GET in body
C. No difference
D. GET is faster

6) What does `<fieldset>` do?
A. Styles form fields
B. Groups related fields semantically; can disable all at once
C. Creates a dropdown
D. Validates form

7) What is FormData used for?
A. Storing form validation rules
B. Collecting and serializing form values for API submission
C. Displaying error messages
D. Styling forms

8) Which is correct for accessibility?
A. `<input type="text" placeholder="Name">`
B. `<label for="name"></label><input id="name">`
C. `<input type="text" name="name">`
D. No label needed

9) What does `aria-invalid="true"` indicate?
A. Field should be ignored
B. Screen readers announce field error state
C. Field is disabled
D. Field is not required

10) Which input type allows file upload?
A. `type="file"`
B. `type="upload"`
C. `type="document"`
D. `type="attachment"`

11) What does `readonly` attribute do?
A. Field cannot be edited but is submitted
B. Field cannot be edited or submitted
C. Field is hidden
D. Field is required

12) Which element groups options in a select dropdown?
A. `<optgroup>`
B. `<group>`
C. `<options>`
D. `<fieldset>`

13) What is `<datalist>` used for?
A. Calendar picker
B. Autocomplete suggestions for text input
C. Date selection
D. Dropdown list

14) What does `novalidate` on form do?
A. Requires manual validation
B. Disables browser native validation
C. Enables strict validation
D. Validates only required fields

15) Which HTML validates automatically without JavaScript?
A. `<input type="text" required>` (empty submission attempt)
B. `<input type="email">` (invalid format check)
C. `<input type="number" min="0" max="100">` (range check)
D. All of the above

16) What is progressive enhancement in forms?
A. Fancy animations
B. Form works without JavaScript; JS adds features
C. Advanced validation only
D. Complex form layouts
